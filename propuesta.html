01 · Scanner tecnológico

El ecosistema
que encontramos

6 herramientas activas. Todas operan en silos. El flujo de datos entre ellas es 100% manual.

ERP · POS
🧾
Bsale
Sistema central de ventas físicas, facturación electrónica, inventario, punto de venta, clientes y reportes. Tiene API REST disponible — la más importante del ecosistema.
API REST ✓ POS físico Facturación electrónica Stock
E-COMMERCE
🛒
Jumpseller
Tienda online en infopet.jumpseller.com. 910 productos cargados uno a uno. Tiene importación CSV nativa y API REST. Actualmente desincronizado del stock físico en Bsale.
API REST ✓ Import CSV ⚠ Sin sync stock
PAGOS
💳
Mercado Pago
Cobro online vía Jumpseller y cobro físico con lector en tienda. Doble canal de pago sin reporte unificado. Los ingresos web y físicos se revisan por separado.
API REST ✓ Cobro web Lector físico
ANÁLISIS · DATOS
📊
Excel / Google Sheets
Carga Maestra Actualizada 2025: fuente de verdad manual para productos. Columnas: Clasificación, Tipo, Nombre, Variante, Marca, Código Barra, SKU, Precios. Actualizada a mano.
⚠ Manual Carga Maestra Gráficos Facturas–Deudas
ARCHIVOS · DOCS
☁️
Google Drive
Repositorio de archivos compartidos. Usado principalmente como carpeta de almacenamiento, no como sistema de datos. Facturas de proveedores, reportes y documentos varios.
API REST ✓ Archivos ⚠ Sin estructura
COMUNICACIÓN
📧
Gmail
Comunicación operacional con clientes y proveedores. Sin automatizaciones activas. Confirmaciones de pedidos, seguimiento y contacto con proveedores se hacen manualmente.
API ✓ Proveedores ⚠ Sin automatizar
02 · Diagnóstico en campo

5 procesos rotos
identificados

Levantados directamente en la tienda con evidencia fotográfica y observación del flujo de trabajo real.

01
Triple ingreso de productos
Carga Maestra → Bsale → Jumpseller · Cada producto se ingresa 3 veces a mano
50%
Tiempo gerente
▾
🔴 Situación actual
Llega mercadería → se ingresa en Google Sheets (Carga Maestra)
Luego se ingresa el mismo producto en Bsale (POS)
Luego se ingresa en Jumpseller (tienda web) con foto y descripción
Los 3 sistemas quedan frecuentemente desincronizados
El gerente dedica el 50% de su tiempo a esta tarea
🟢 Con la solución
Un solo formulario → dispara a los 3 sistemas simultáneamente
Bsale API + Jumpseller API + Google Sheets Append en paralelo
Tiempo estimado por producto: de 20 min → 2 min
Carga masiva desde CSV con precheck antes de enviar
02
Facturas de proveedor sin digitalizar
80% digital (no descargable en Bsale) · 20% solo en papel físico
5h
Perdidas/semana
▾
🔴 Situación actual
Facturas digitales en Bsale aparecen en azul (visibles) o gris (no descargables)
20% de facturas solo existen en papel físico
Para ver el detalle de productos hay que tener el papel a mano
No hay forma de buscar o filtrar facturas antiguas eficientemente
🟢 Con la solución
Factura digital: extrae líneas via Bsale API automáticamente
Factura física: foto desde celular → Claude Vision OCR → extrae datos
Formulario pre-llenado para revisar y confirmar
Imagen archivada en Google Drive con nombre estructurado
03
Stock físico desincronizado del stock web
Bsale y Jumpseller no se comunican · Riesgo real de sobreventa online
⚠
Riesgo diario
▾
🔴 Situación actual
Venta en tienda física (Bsale) no descuenta stock en Jumpseller
Cliente puede comprar online un producto agotado en tienda
Los precios pueden diferir entre Bsale y Jumpseller
No hay visibilidad unificada de lo que se vende por canal
🟢 Con la solución
Webhook Bsale → Vercel Function → actualiza stock en Jumpseller
Dashboard que muestra stock físico vs web en paralelo
Alerta cuando hay diferencia mayor al 10%
Reporte unificado de ventas físicas + web
04
Sin visibilidad de stock crítico para compras
No existe un sistema que indique qué comprar, cuánto y a quién
$M
Costo en quiebres
▾
🔴 Situación actual
No hay dashboard de compras urgentes vs posibles
Decisión de compra basada en memoria o revisión manual
No hay cálculo automático de cuánto pedir por producto
El proveedor y costo unitario no están asociados al stock en ningún sistema
🟢 Con la solución
Dashboard semáforo: 🔴 Urgente / 🟡 Posible / 🟢 Full
Calcula automáticamente cantidad a pedir (stock ideal – stock actual)
Muestra costo estimado de reposición por producto y total
Genera orden de compra exportable en 1 clic
05
Datos en Excel sin conexión en tiempo real
Reportes, gráficos y listas de precios actualizados manualmente
3h
Semanales
▾
🔴 Situación actual
Exportar de Bsale → pegar en Excel → subir a Drive cada vez
Múltiples versiones del mismo archivo en Drive
Lista de precios en Excel puede estar desactualizada respecto a Bsale
Gráficos de consultas y dispositivos no tienen fuente de datos clara
🟢 Con la solución
Dashboard en vivo conectado a Bsale API — sin exportar
Gráficos de ventas, stock, consultas actualizados en tiempo real
Lista de precios exportable desde la app en 1 clic
Drive queda solo para archivos que realmente lo requieren
